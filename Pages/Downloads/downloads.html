Prism OS

Prism OS is the latest operating system that is being actively developed by a small group of developers. The idea started during the beginning of the Covid pandemic during the lockdowns. While it was on GitHub, more people eventually started contributing and helping develop the OS. The OS later started getting a little more support from some friends who helped develop the OS. As time gone on, the projected started growing and becoming more than a hobby but instead a real project with real potential. In fact, we somehow managed to get the main developer of Linux to commit on the project!

Lemon OS

Lemon OS was the original name that was chosen for this project. Later on, it was decided to be renamed to PrismOS. Lemon OS was the original operating system which was made by Terminal.cs (Original Developer) which was later posted on GitHub for other people to discover. It was simple made for fun without the intention to become something more than some fun side project.
